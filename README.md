# Movie Review API

## Description

GraphQL API of the Movie review website using Apollo Server, Prisma and Nexus

## Feature

1. Manage review, rating
2. Wrap TMDB api
3. Set HttpOnly Cookie

## Stack

1. Typescript
2. Apollo Server
3. Prisma
4. Nexus
5. MySQL
