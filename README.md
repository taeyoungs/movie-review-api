# Movie Review API

GraphQL API of the Movie review site using Apollo, Prisma

## Feature

1. See Popular, Upcoming movies
   (Using TMDB rest api)
2. Token auth (google, github)
3. crud review and comment

<br />

## Stack

1. Typescript
2. Apollo
3. Prisma 2
4. Nexus
5. Express
6. Nodejs
7. MySQL
